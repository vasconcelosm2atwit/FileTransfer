********************* HOW TO ADD NEW FILES???? *******************************

TO SHARE FILES TO CLIENT:
	ADD ALL FILES YOU WISH TO SHARE TO THE "SHARE" FOLDER SO THE CLIENT MAY ACCESS AND DOWNLOAD THE FILES

Make sure the application is CLOSED when new files are being added to the folder or 
it wont show for the client
